# 图书编辑 (book-editor)

## 概述
本技能专门用于审核本书中文版的 Quarto Markdown (`.qmd`) 内容。它严格执行 `GEMINI.md` 中定义的翻译、排版和交叉引用规范。

## 核心审核清单

### 1. 术语翻译 (Terminology)
- **格式**：专业术语首次出现或定义时，必须使用 **中文加粗** (*英文斜体*)。
- **一致性**：检查常用术语（如：**最大似然估计量** (*maximum likelihood estimator*)）在全书中的统一性。

### 2. 标题与锚点 (Headers & Anchors)
- **章节主文件**：一级标题应翻译为中文，并保留锚点（如：`# 寿命测试 {#sec-14}`）。
- **子章节**：小节标题使用 `##`，且必须紧跟 ID 锚点（格式：`{#sec-章节号_小节号}`）。
- **注意**：锚点与标题文字之间应有一个半角空格。

### 3. 交叉引用 (Cross-References)
- **禁止纯文本引用**：不得使用“示例 1.1”等文字引用。
- **自动引用语法**：必须使用 `@sec-ID`、`@eq-ID`、`@thm-ID`、`@def-ID`、`@cor-ID`、`@exr-ID`（示例）、`@sol-ID`（解答）、`@tbl-ID`（表格）、`@fig-ID`（图形）。

### 4. 专用块 ID (Special Blocks)
- **示例与解答**：必须包裹在 Quarto 自定义块中，并带有 ID。
  - 示例：`::: {#exr-章节号_小节号_序号}`
  - 解答：`::: {#sol-章节号_小节号_序号}`

### 5. 数学公式 (Math Formatting)
- **定界符**：行外公式的 `$$` 必须各占一行。
- **对齐环境**：复杂推导必须使用 `\begin{align} ... \end{align}`，且必须包裹在 `$$ ... $$` 中。
- **间距**：中文与行内公式 `$ ... $` 之间必须保持一个半角空格。
- **标点**：数学环境（`$...$` 或 `$$...$$`）内使用西文标点。

### 6. 绘图规范 (R Graphics)
- **标签语言**：R 代码生成的图形标签、轴标题必须保持 **英文**。
- **图形引用**：必须配置 `#| label: fig-ID` 属性。

## 审核工作流

1. **扫描文件**：对请求审核的文件进行逐行扫描。
2. **识别违规**：标记不符合上述规范的行。
3. **OCR 校验**：特别检查容易混淆的中文同音字（如：贝叶斯/贝育）。
4. **生成报告**：简述发现的问题，并提供修改建议。

## 常见错误（反面示例）
- ❌ `示例 1.1 显示了...` -> ✅ `@exr-1_1_1 显示了...`
- ❌ `$$y = ax + b$$` -> ✅ `$$ \n y = ax + b \n $$` (定界符需换行)
- ❌ `贝育斯分布` -> ✅ `贝叶斯分布`
- ❌ `...变量$x$的正态分布...` -> ✅ `...变量 $x$ 的正态分布...` (缺少半角空格)
